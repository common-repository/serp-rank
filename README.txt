== Description ==

Keyword Rank Tracker helps sites owner to track Google's keywords position of pages in search results. 

== Installation ==

1. Upload `serp-rank` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Keyword Rank Tracker Settings page and follow steps shown on this page to configure plugin.
4. Navigate to Pages or Posts page to view current positions.

== Frequently Asked Questions ==


= How do I install Keyword Rank Tracker? =

Here are the installation guidelines: <a href="https://www.youtube.com/watch?v=Av_tbF7QWyA&t=65s">https://www.youtube.com/watch?v=Av_tbF7QWyA</a> 
 
= I want to refresh cache =
 
Navigate to Keyword Rank Tracker settings page and set Cache lifetime = 1 second. Refresh Pages or Posts page. You will see updated data. Then navigate to settings page again and return previous value (86400 by default)
 
= I don't see any rank column on my Posts or Pages list  =
 
Navigate to Keyword Rank Tracker settings page and make you have "Authenticated" status message on top of the page. If you don't - follow steps on this page.
 
= How do I set up Excluded keywords? =
 
Navigate to Keyword Rank Tracker settings page and scroll down to Excluded Keywords. Then write your Keyword down that you want to Exclude.
